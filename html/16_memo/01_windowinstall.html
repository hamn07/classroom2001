  2. nodejs (為了讓sbulime的HTML-CSS-JS Prettify套件可以運行)
  3. sublime (挑選符合自己的作業系統版本)
    安裝sublime
    安裝package control (記得要選擇頁面上的sublime text3) (如果當貼上有問題，請試試其他瀏覽器)
    View → Show Console ，打開Console，把上一個步驟的指令碼貼上執行
    Preferences → Package Control → 輸入Install Package
    搜尋 HTML-CSS-JS Prettify 安裝 (讓程式碼可以排版的套件)
    再重新打開Preferences → Package Control → 輸入Install Package
    搜尋 ConvertToUTF8 (排除中文亂碼)
    如果要刪除package → Preferences → Package Control → 輸入 Remove Pacekage ...
