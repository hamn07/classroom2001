網頁瀏覽或執行，是開啟在瀏覽視窗（window）
包含了一份的HTML文件（document）
文件中又包含了各種物件（object）
每個物件都可以加以命名（name）或加上識別碼（id）

物件？物件導向？
String （類別）裡面包含了 length （屬性） 和 indexOf()、substr()、split()等（方法、凾式）
Image （類別）裡面包含了 src （屬性） 和 onload（事件）

對JavaScript來說，物件是 屬性、方法、事件的集合

DOM：Document Object Model
文件、物件、模型
DOM不是一種技術，只是存取結構化文件的一種思想 一份有結構化的文件，透過DOM模型，轉換成DOM樹，讓我們可以用物件的方式操作html元素
